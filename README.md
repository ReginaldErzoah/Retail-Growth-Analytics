# Retail Growth Analytics Dashboard

## **Project Overview**

This project showcases a **Retail Growth Analytics dashboard** built using the Sample Metabase dataset. It provides actionable insights into **customer acquisition, engagement, retention, product performance, and revenue trends**, simulating a Growth Analyst’s work in retail or banking.

---

## **Key Features**

* **New Accounts & Users Over Time** – Track acquisition trends.
* **Activation & Conversion Funnel** – Measure signup > first event > first order.
* **Revenue & Product Performance** – Identify top products and analyze customer ratings.
* **Orders & Engagement Trends** – Monitor repeat purchases and user activity.

---

## **Tools & Technologies**

* H2 Database, SQL
* Metabase (interactive dashboard)
* Data aggregation, funnel, and trend analysis
